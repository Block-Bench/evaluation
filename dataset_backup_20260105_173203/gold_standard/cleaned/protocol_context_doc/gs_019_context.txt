PROTOCOL CONTEXT: Gearbox - Midas Redemption Adapter

PROTOCOL TYPE: DeFi Integration / Protocol Adapter

OVERVIEW:
MidasRedemptionVaultAdapter is the Gearbox adapter for interacting with Midas vault redemptions. It translates Gearbox credit manager calls into Midas gateway operations, and manages phantom tokens that represent pending redemption claims within the credit account system.

BUSINESS MODEL:
- Provides interface for credit accounts to request and claim Midas redemptions
- Manages phantom tokens that track pending redemption value
- Phantom tokens are used for collateral calculations while redemptions are pending
- Coordinates between credit manager, gateway, and phantom token contracts
- Handles multiple output token types for redemption flexibility
