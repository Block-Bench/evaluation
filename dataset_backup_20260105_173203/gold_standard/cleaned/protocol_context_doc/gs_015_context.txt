PROTOCOL CONTEXT: Aragon - Lock-to-Vote Plugin

PROTOCOL TYPE: DAO Governance / Voting Plugin

OVERVIEW:
LockToVotePlugin is the main governance plugin for the Aragon Lock-to-Vote system. It handles proposal creation, vote tallying, and proposal execution. It defines what actions a proposal can take and manages the proposal lifecycle.

BUSINESS MODEL:
- Authorized users create proposals with a set of executable actions
- Proposals have configurable parameters (voting duration, thresholds, etc.)
- Token holders vote on proposals through the LockManager
- Proposals are executed when voting ends successfully
- The plugin can execute arbitrary actions on behalf of the DAO
